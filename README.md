#　一拖即存

一拖即存的使用非常简单，就象用网络蚂蚁一样，只需用鼠标将我们要保存的内容选中，然后用鼠标拖到“一拖即存”的拖存窗口一放，程序将自动地根据你的设置，在后台将所选内容保存为一个文件，放在你想要放的分类目录里。同时支持“断电重启自动恢复”、“GB－BIG5内码转换”、“段落重排”、“多文本合并”、“程序换肤”等多种功能，完全免费。
